Python Basics

Welcome to Python! Python is a popular high-level, interpreted programming language known for its readability and simplicity. Below are the foundational concepts every Python beginner should learn:

1. Hello World

Your first Python program:

print("Hello, World!")

2. Variables and Data Types

Variables are used to store data.

name = "Alice"
age = 25
height = 5.6
is_student = True
  • String: Text data (e.g., "Hello")
  • Integer: Whole numbers (e.g., 25)
  • Float: Decimal numbers (e.g., 5.6)
  • Boolean: True or False values

3. Conditional Statements

Used to make decisions in code:

if age > 18:
    print("Adult")
else:
    print("Minor")

4. Loops

Repeat tasks using loops:

For Loop 
for i in range(5):
    print(i)
While Loop 
count = 0
while count < 5:
    print(count)
    count += 1

5. Functions

Reusable blocks of code:

def greet(name):
    print(f"Hello, {name}!")

6. Lists and Tuples

Used to store collections:

fruits = ["apple", "banana", "cherry"]
coordinates = (10.0, 20.0)

7. Dictionaries

Key-value pairs:

person = {"name": "Alice", "age": 25}

8. Input from User

Get data from the user:

name = input("Enter your name: ")
print("Hello, " + name)

9. Exception Handling

Handle errors gracefully:

try:
    x = int(input("Enter a number: "))
except ValueError:
    print("That's not a number!")

10. Object-Oriented Programming (OOP)

Create reusable blueprints using classes:

class Student:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print(f"Hi, I'm {self.name}")

s1 = Student("Alice")
s1.greet()
